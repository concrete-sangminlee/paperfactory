# PaperFactory

AI-powered academic paper generator for structural engineering research.

PaperFactory automates the full research-to-paper pipeline: **literature review в†’ research design в†’ code execution в†’ result analysis в†’ paper writing**. It generates journal-formatted Word documents targeting top structural engineering journals.

## Supported Journals

| Journal | Publisher |
|---------|-----------|
| ASCE Journal of Structural Engineering | ASCE |
| ACI Structural Journal | ACI |
| Journal of Wind Engineering and Industrial Aerodynamics | Elsevier |
| Journal of Building Engineering | Elsevier |
| Engineering Structures | Elsevier |

## How It Works

1. **Enter a research topic** вЂ” e.g., "Deep learning-based seismic damage detection in RC structures"
2. **Select a target journal** вЂ” guidelines are automatically applied
3. **Pipeline runs in 5 steps:**
   - Literature Review (web search + gap analysis)
   - Research Design (methodology + experiment plan)
   - Code Execution (generates & runs Python research code)
   - Result Analysis (scholarly interpretation)
   - Paper Writing (full paper + Word export)
4. **Review each step** or run in full auto mode
5. **Download** the formatted `.docx` file

## Requirements

- **Python 3.10+**
- **[Claude Code CLI](https://docs.anthropic.com/en/docs/claude-code)** installed and authenticated
  - Requires an active Claude Code subscription (Max Plan recommended)

## Configuration

Journal guidelines are stored as JSON in `guidelines/`. You can edit these files to fine-tune formatting, reference styles, or section structures.

## Disclaimer

Papers generated by PaperFactory are AI-assisted drafts. They require thorough human review, verification, and refinement before submission. Always validate research results, citations, and claims independently.
